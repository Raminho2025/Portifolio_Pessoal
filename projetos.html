Visite o Meu Projeto Hamburgueria - "A Caseira"

Projeto Hamburgueria Artesanal - "A Caseira" O projeto é um site responsivo desenvolvido para uma hamburgueria artesanal fictícia. O objetivo do site é apresentar o cardápio, destacar os diferenciais da marca, permitir contato direto com os clientes e garantir uma experiência fluida tanto em dispositivos móveis quanto em desktops. O layout segue um estilo moderno com paleta em tons quentes (vermelho), fontes robustas e imagens em alta definição dos lanches. A navegação é simples e intuitiva, com uma página principal que é Cardápio. Com um botão que muda de cor representando o estabelecimento (verde para aberto e vermelho para fechado) em JavaScript. O projeto utiliza HTML5, CSS3 com Flexbox e Grid, além de JavaScript puro para a lógica de interação. Também aplica princípios básicos de SEO e boas práticas de acessibilidade, como contraste adequado e uso de tags semânticas.
